Evaluation Metrics for Healthcare Machine Learning Models
When evaluating machine learning models in healthcare, especially for diagnostic or prognostic tasks, multiple metrics are needed because no single measure captures all desirable properties pmc.ncbi.nlm.nih.gov+1.

1. Core Classification Metrics
For binary classification (e.g., disease vs. no disease), common metrics include:

Accuracy: Overall proportion of correct predictions. Can be misleading if classes are imbalanced.

Precision: Proportion of true positives among predicted positives. Important when false positives are costly.

Recall (Sensitivity): Proportion of true positives correctly identified. Important when false negatives are costly.

F1 Score: Harmonic mean of precision and recall. Balances both when they are in conflict.

Specificity: Proportion of true negatives correctly identified.

Matthews Correlation Coefficient (MCC): Robust to imbalanced datasets.

2. Discrimination Metrics
Measure how well the model distinguishes between classes:

Area Under the ROC Curve (AUC-ROC): Overall ability to discriminate; 0.5 = random, 1.0 = perfect The Lancet.

Area Under the Precision-Recall Curve (AUC-PR): More informative when classes are imbalanced.

3. Calibration Metrics
Assess how well predicted probabilities match observed outcomes:

Calibration Plot: Visualizes predicted vs. actual probabilities.

Brier Score: Mean squared difference between predicted and actual probabilities.

4. Clinical Utility Metrics
Go beyond statistical performance to decision-making impact:

Net Benefit: Measures the net gain in correct decisions over a threshold.

Decision Curve Analysis (DCA): Plots net benefit across thresholds to show clinical usefulness The Lancet.

Probability Distribution Plots: Show how well the model’s outputs reflect real-world prevalence.

5. Specialized Metrics for Healthcare
Negative Predictive Value (NPV): Useful when false negatives are critical.

Positive Predictive Value (PPV): Useful when false positives are critical.

Likelihood Ratios: Help interpret model outputs in diagnostic contexts.
6. Reporting Best Practices
Report multiple metrics to capture different aspects of performance pmc.ncbi.nlm.nih.gov.

Use proper scoring rules (e.g., Brier score) to avoid bias from improper measures The Lancet.

Include calibration and clinical utility plots alongside statistical metrics.

Consider threshold effects: Many classification metrics depend on the decision threshold; report performance at clinically relevant thresholds.
