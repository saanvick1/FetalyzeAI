## Overview

**FetalyzeAI** is an explainable AI system for classifying fetal health from Cardiotocography (CTG) signals. Each year, over **130 million babies** are born worldwide — and up to **30% of neonatal deaths** are linked to undetected intrapartum fetal distress.

This project trains and evaluates on **real intrapartum waveform data** from the CTU-UHB and CTGDL databases — raw FHR/UC signals from real hospital recordings, not tabular feature extracts.

---

## Datasets Used

| Dataset | Size | Purpose |
|---------|------|---------|
| **CTU-UHB Intrapartum CTG (PRIMARY)** | 552 real intrapartum CTG recordings | Primary model training & evaluation |
| **CTU-CHB Annotation Dataset (CTGDL)** | Expert morphological annotations on CTU-UHB | Event / explainability engine |

- **CTU-UHB**: FHR + UC at 4 Hz, up to 90 min before delivery. Outcomes: cord blood pH, base deficit, Apgar scores. License: ODC-BY-1.0. Source: [PhysioNet](https://physionet.org/content/ctu-uhb-ctgdb/1.0.0/)
- **CTGDL**: Zenodo-hosted preprocessed CSVs and FHRMA annotation archives. Source: [Zenodo 19510407](https://zenodo.org/records/19510407)

## Key Results (CTU-UHB Primary Model)

| Metric | Value |
|--------|-------|
| Held-out test F1 (macro) | primary metric |
| High-risk recall | clinical priority |
| AUROC (macro) | primary metric |
| Architecture | Regularized XGBoost (depth 3) + 5-member MLP ensemble + temperature scaling |

Run `python train_ctu_signal_model.py` to generate real evaluation numbers into `ctu_model_results.json`.

---

## Architecture

- **XGBoost** (depth 3, strong L1/L2 regularization, class-balanced)
- **5-member MLP ensemble** with temperature scaling for calibrated uncertainty
- **Signal feature extraction** from raw FHR/UC waveforms (not tabular features)
- **Leakage-free pipeline**: imputer/scaler fit only on training indices inside each fold

## Scientific Hypotheses

**Primary Research Question:** Can a hybrid neural architecture trained on real intrapartum CTG waveforms achieve clinically reliable fetal distress classification?

**H₁ (Alternate):** FetalyzeAI trained on CTU-UHB waveforms achieves >90% high-risk recall and >0.90 AUROC on held-out CTU-UHB test data.

**H₀ (Null):** The model does not achieve clinically meaningful performance on real intrapartum data.
